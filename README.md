# FILE-EdoTensei

### 개요
>  문제 정의

파일 시스템(NTFS, FAT32)의 MFT(Master File Table)나 FAT(File Allocate Table) 영역이 손상되었을 때, OS는 파일을 찾지 못한다. 

> 목표

파일 시스템의 메타데이터 및 인터페이스에 의존하지 않고, 저장 매체의 Raw Data를 직접 분석하여 삭제된 파일에 해당하는 데이터 영역을 카빙 방식으로 재구성하는 디지털 포렌식 도구를 개발한다.

### 핵심 기능

> Disk I/O

> 지원 포맷
